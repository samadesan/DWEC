Ejercicio 1 - AJAX con PHP + Base de Datos: Consulta dinámica en tiempo real

Realiza una página web que ejecute el contenido de un programa que realizarás en php interactuando con una Base de Datos que crearás previamente. El programa es libre y la BD también. Ejecuta algo similar a los ejemplos vistos en clase.

Ejercicio 2 - Carga de archivo PHP mediante AJAX usando POST

Realiza una página web con el método POST para traer un fichero php.

Ejercicio 3 - Envío de parámetros con AJAX POST y respuesta dinámica

Realiza una página web con el método POST utilizando parámetros.

Ejercicio 4 - Intercambio de datos vía AJAX utilizando XML

Realiza una página web con un fichero XML para envío de datos

Ejercicio 5 - Intercambio de datos vía AJAX utilizando JSON

Realiza una página web con un fichero JSON para envío de datos
