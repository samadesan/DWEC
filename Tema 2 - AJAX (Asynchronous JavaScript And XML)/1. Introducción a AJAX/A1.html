Ejercicio 1 - Mostrar Fichero de Texto en Ajax

Realizar un ejercicio similar al ejemplo guía para mostrar los datos de un fichero de texto.txt

Ejercicio 2 - Cargar de forma asíncrona los enlaces pinchados

Realizar un ejercicio con dos enlaces a dos páginas web para que cada vez que se pinche en uno de los enlaces se cargue la página correspondiente de forma asíncrona. Las páginas html deben estar alojadas en el mismo lugar que la principal. No deben tener cabecera HTML, HEAD, TITLE ya que se incluyen como contenido dentro de otra página web. En este ejemplo que os visualizo al pinchar sobre inicio sale la página de inicio.html, cuyo código es el siguiente:

html

<h2 class="aligncenter">INICI</h2>
    <p class="aligncenter">
        <em>Zona de continguts</em>
    </p>

Ejercicio 3 - Cambio asíncrono de imagen con botón.

Realizar una página web que contenga una imagen que se cambie por otra de forma asíncrona (por ejemplo al pinchar un botón “cambio de imagen” se debe producir el cambio).

Ejercicio 4 - Visualización asíncrona de un fichero XML.

Realizar una página web que muestre el contenido de un fichero XML de forma asíncrona (por ejemplo al pinchar en un botón “Mostrar fichero XML” se debe mostrar el contenido)

Ejercicio 5 - Ejecución dinámica de JavaScript externo con eval.

Realizar una página web que haga una llamada a un fichero JavaScript(extension .js) y ejecute la respuesta correctamente(utiliza la función eval).

Ejercicio 6 - Ejecución de programa PHP sin parámetros.

Realiza una página web que ejecute el contenido de un programa que realizarás en php. El programa es libre, por ejemplo un programa para mostrar la fecha de sistema.(En este ejercicio no uses parámetros)

Ejercicio 7 - Ejecución de programa PHP con parámetros vía GET.

Realiza una página web que ejecute el contenido de un programa que realizarás en php. El programa es libre, pero deber interactuar con los parámetros que le pasaremos con la orden GET.(En este ejercicio debes usar parámetros)

Ejercicio 8 - Visualización asíncrona de fichero JSON.

Realiza una página web que muestre el contenido de un fichero JSON de forma asíncrona(por ejemplo al pinchar un botón “Mostrar fichero JSON” se debe mostrar el contenido).

Ejercicio 9 - Indicador de carga AJAX con logo animado.

Realiza una página web que muestre un logo típico de la espera de información en Ajax(ej. Gif animado) mientras estamos esperando que nos devuelva algún dato con php.
