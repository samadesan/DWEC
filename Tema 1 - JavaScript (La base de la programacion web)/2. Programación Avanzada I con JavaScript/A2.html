Ejercicio 1 - Calcular el iva y el precio total(con el iva incluido) de un artículo

Introducir el precio de un artículo. Calcular el iva y el precio total(con el iva incluido) que debe salir redondeado a dos decimales. Debéis utilizar una función.

Ejercicio 2 - Introducir un mes y decir a qué estación pertenece. Con switch

Introducir un mes y decir a qué estación pertenece

Ejercicio 3 - Mostrar alert para continuar

Crear una página web que nos muestre el mensaje “Bienvenido a mi página” con formato H1, nos muestre la plataforma del navegador que estamos utilizando, y con la ventana de Confirm nos pida si queremos continuar. En caso de que así sea saldrá en pantalla el mensaje “continuamos…”

Ejercicio 4 - Redimensionar 500x500 según el navegador

Crear un programa que nos permita redimensionar la pantalla con el método resize a 500x500 en caso de que sea Internet Explorer, si es otro navegador que salga el mensaje “Operación no permitida con este navegador”

Ejercicio 5 - Fecha Actual

Crear una página con el botón “Fecha Actual”. Al pinchar sobre el botón debe salir la fecha actual del sistema pero con el día y el mes en letra. Ejemplo:

 Lunes, 15 de Octubre de 2012

Ejercicio 6 - Hora Actual

Crear una página con el botón “Hora Actual”. Al pinchar sobre el botón debe salir un mensaje del tipo:

 Buenos días, son las 12:30 horas

El saludo debe variar según la hora del día. Buenos días hasta las 14:00 horas, Buenas tardes hasta las 20:00 y a partir de esta hora debe salir Buenas noches

Ejercicio 7 - Selector de cambio de fondo

Crea una página con tres botones con el texto Azul, Verde y Rojo. Al pinchar sobre cada botón debe cambiar el fondo de la página al color indicado. Utilizaremos el evento onclick, en temas posteriores veremos otros eventos

Ejercicio 8 - Datos de la página web

Mostrar en pantalla un documento que nos muestre:

  • La URL del documento actual
  • El pathname de la página (corresponde a la cadena que sigue al nombre del servidor).
  • El protocolo utilizado por la página web.

Debéis insertar también un botón para que al pinchar sobre él se cargue la página de Google

Ejercicio 9 - Abrir otra página distinta

Crear una página que contendrá el botón Abrir . Al pinchar sobre el botón debe aparecer otra ventana que contendrá, a su vez, el botón Cerrar. Al pinchar sobre él debemos volver a la página inicial.
