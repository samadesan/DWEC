Ejercicio 1 - Formulario de 3 campos con campo no vacío obligatorio (primera forma)

Crear un formulario con tres campos. Cancela el envío del mismo mientras haya algún campo vacío Muestra un mensaje de la cancelación del envío y otro cuando este se produzca.

Ejercicio 2 - Formulario de 3 campos con campo no vacío obligatorio (segunda forma)

Crea un documento con un link. Evita que se produzca el comportamiento normal y que al hacer clic sobre el enlace este no vaya al sitio indicado.
