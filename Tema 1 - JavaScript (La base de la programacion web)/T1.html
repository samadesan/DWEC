Capítulo 1. Programación básica

1.1. Variables

Una variable es un identificador que almacena un valor. Permiten crear programas genéricos que funcionan con distintos datos. En javascript se suelen declarar con var, aunque también es posible usarlas sin declarar, generando variables globales, algo no recomendable.

1.1.1. Nombres de variables

Solo pueden contener letras, números, $ y _, pero no pueden comenzar por un número.

1.2. Tipos de variables

1.2.1. Numéricas

Almacenan números enteros o decimales. Los decimales usan punto. Ejemplo:

js 

var total = 25.4;
1.2.2. Cadenas de texto

Guardan texto usando comillas simples o dobles. Para incluir caracteres especiales se utiliza el mecanismo de escape: \n, \t, \', \", \\.

1.2.3. Arrays

Son colecciones de valores dentro de corchetes. Los elementos se numeran desde la posición 0. Ejemplo:

js 

var dias = ["lunes","martes","miércoles"];
1.2.4. Booleanos

Solo pueden almacenar true o false. Se usan para decisiones lógicas dentro del programa.

1.3. Operadores

1.3.1. Asignación

El operador = asigna un valor a una variable. La asignación siempre debe hacerse sobre una variable, nunca sobre un literal.

1.3.2. Incremento y decremento

Los operadores ++ y -- modifican en una unidad el valor de una variable numérica. Pueden colocarse antes o después del nombre de la variable, cambiando cuándo se aplica el incremento o decremento.

1.3.3. Operadores lógicos

Permiten tomar decisiones en función de condiciones. ¡El operador de negación "!", invierte el valor lógico. Los números se convierten a booleanos siendo el 0 igual a "false" y cualquier otro número true; en cadenas, ("") es false y cualquier otra es +true.

Capítulo 2. Programación avanzada

Las estructuras de control, los operadores y todas las utilidades propias de JavaScript que se han visto en los capítulos anteriores, permiten crear scripts sencillos y de mediana complejidad. Sin embargo, para las aplicaciones más complejas son necesarios otros elementos como las funciones y otras estructuras de control más avanzadas, que se describen en este capítulo.

2.1. Funciones

Cuando se desarrolla una aplicación compleja, es muy habitual utilizar una y otra vez las mismas instrucciones. Un script para una tienda de comercio electrónico por ejemplo, tiene que calcular el precio total de los productos varias veces, para añadir los impuestos y los gastos de envío.

Cuando una serie de instrucciones se repiten una y otra vez, se complica demasiado el código fuente de la aplicación, ya que:

  • El código de la aplicación es mucho más largo porque muchas instrucciones están repetidas.
  • Si se quiere modificar alguna de las instrucciones repetidas, se deben hacer tantas modificaciones como veces se haya escrito esa instrucción, lo que se convierte en un trabajo muy pesado y muy propenso a cometer errores.

Las funciones son la solución a todos estos problemas, tanto en JavaScript como en el resto de lenguajes de programación. Una función es un conjunto de instrucciones que se agrupan para realizar una tarea concreta y que se pueden reutilizar fácilmente.

Ejemplo de código repetido:

js 

var resultado;
var numero1 = 3;
var numero2 = 5;
resultado = numero1 + numero2;
alert("El resultado es " + resultado);

numero1 = 10;
numero2 = 7;
resultado = numero1 + numero2;
alert("El resultado es " + resultado);

numero1 = 5;
numero2 = 8;
resultado = numero1 + numero2;
alert("El resultado es " + resultado);

Función para agrupar instrucciones:

js 

function suma_y_muestra() {
    resultado = numero1 + numero2;
    alert("El resultado es " + resultado);
}
2.1.1. Argumentos y valores de retorno

Funciones con argumentos:

js 

function suma_y_muestra(primerNumero, segundoNumero) {
    var resultado = primerNumero + segundoNumero;
    alert("El resultado es " + resultado);
}

var numero1 = 3;
var numero2 = 5;
suma_y_muestra(numero1, numero2);

Función para calcular precio total con return:

js 

function calculaPrecioTotal(precio, porcentajeImpuestos) {
    var gastosEnvio = 10;
    var precioConImpuestos = (1 + porcentajeImpuestos/100) * precio;
    var precioTotal = precioConImpuestos + gastosEnvio;
    return precioTotal.toFixed(2);
}

var precioTotal = calculaPrecioTotal(23.34, 16);
var otroPrecioTotal = calculaPrecioTotal(15.20, 4);

2.2. Ámbito de las variables

El ámbito (scope) de una variable es la zona del programa en la que está definida. En JavaScript existen dos ámbitos: global y local.

Ejemplo:

js 

function creaMensaje() {
    var mensaje = "Mensaje de prueba";
}
creaMensaje();
alert(mensaje); // No funciona: mensaje es local

Capítulo 3. DOM (Document Object Model)

El DOM es la estructura en forma de árbol que crea el navegador al interpretar un documento HTML. Gracias a esta estructura, JavaScript puede acceder, modificar, crear o eliminar cualquier parte de una página web mientras está cargada.

3.1. Árbol de nodos

Cuando el navegador carga una página, convierte el HTML en una estructura de nodos. Cada etiqueta, texto o comentario se convierte en un nodo dentro de un árbol jerárquico.

Ejemplo de HTML:

html 

<p>Esta página es <strong>muy sencilla</strong></p>

Esto produce los siguientes nodos:

  • Nodo Element: <p>
  • Nodo Text: "Esta página es "
  • Nodo Element: <strong>
  • Nodo Text: "muy sencilla"

3.2. Tipos de nodos

Los principales tipos de nodos del DOM son:

  • Document: nodo raíz del documento.
  • Element: representa etiquetas HTML.
  • Attr: representa atributos.
  • Text: contenido textual.
  • Comment: comentarios HTML.

3.3. Acceso directo a nodos

JavaScript permite localizar elementos del documento sin recorrer manualmente todo el árbol mediante funciones de acceso directo.

3.3.1. getElementsByTagName()
js 

var parrafos = document.getElementsByTagName("p");

Devuelve un array con todos los elementos del tipo indicado.

3.3.2. getElementsByName()
js 

var elementos = document.getElementsByName("especial");

Busca elementos cuyo atributo name coincida.

3.3.3. getElementById()
js 

var cabecera = document.getElementById("cabecera");

Selecciona un único elemento mediante su identificador único.

3.4. Crear y eliminar nodos

3.4.1. Crear nodos
js 

var p = document.createElement("p");
var texto = document.createTextNode("Hola Mundo!");
p.appendChild(texto);
document.body.appendChild(p);

Al crear un elemento, primero se crea la etiqueta, luego el texto y finalmente se insertan en el documento.

3.4.2. Eliminar nodos
js 

var elemento = document.getElementById("provisional");
elemento.parentNode.removeChild(elemento);

Para eliminar un nodo, se elimina desde su nodo padre.

3.5. Acceso a atributos XHTML

Los atributos HTML se pueden leer y modificar mediante propiedades del nodo.

js 

var enlace = document.getElementById("enlace");
alert(enlace.href);
alert(enlace.id);
Acceso a estilos CSS
js 

var imagen = document.getElementById("imagen");
alert(imagen.style.margin);

Para propiedades CSS con guiones, se utiliza la notación camelCase:
Ejemplo: font-weight → fontWeight
